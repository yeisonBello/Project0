Project 0 - Revature

Description

This is console-based program that simulate some basic banking operations and is conected with a database.  
the main method is in the file BankPrototype.
the database description is in the .txt "DatabaseDescription". 

What can you do ?

as a user :
*create a new Account with a starting balance.
*login.
*view the balance.
*withdrawal or deposit.
*transfer money/ Accept or decline tranfers

as a employee :
*approve or reject an account.
*view customers bank accounts.
*view a log of all transactions.
